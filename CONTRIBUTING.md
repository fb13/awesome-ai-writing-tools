# Contributing to Awesome AI Writing Tools

Thank you for your interest in contributing! This list is community-driven, and every quality submission helps writers discover better tools.

## How to Add a Tool

1. **Fork** this repository.
2. **Create a branch** for your addition (e.g., `add-toolname`).
3. **Add the tool** to the correct category in `README.md`, at the **end** of that category's list.
4. **Submit a pull request** using the provided PR template.

## Entry Format

Each entry must follow this exact format:

```
- [Tool Name](https://example.com/) - One-sentence description ending with a period.
```

- The description should be a single sentence, concise and factual.
- End the description with a period.
- Do not use promotional superlatives ("best", "amazing", "#1").

## Rules

- **One tool per PR.** If you want to add multiple tools, open separate PRs.
- **Add to the end** of the relevant category. Do not insert in the middle.
- **No duplicates.** Search the list first to make sure the tool is not already included.
- **Valid links only.** The tool must be publicly accessible at the provided URL.
- **Correct category.** If no existing category fits, propose a new one in your PR description.
- **No affiliate or referral links.** Use clean, direct URLs.
- **No unrelated changes.** Keep your PR focused on the tool addition.

## Proposing a New Category

If a tool does not fit any existing category, you may propose a new one. In your PR description, explain why the new category is needed and suggest where it should appear in the table of contents.

## Reporting Broken Links or Removing Tools

If you find a tool with a dead link or one that is no longer available, please open an issue or submit a PR to remove it.

## Code of Conduct

Be respectful and constructive. This is a shared resource for the writing community.
